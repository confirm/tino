Typr
====

This repository contains Typr – the web-based Typst editor.
You'll find…

- the [source code](confirm/noddl/),
- the [documentation](docs/),
- the [Dockerfiles](Dockerfiles/) for the `typr/typr`, and the `typr/docs` images,
- and all the required files to build the package, and Docker images.

For more information about Typr, check out the deployed [documentation](https://typr.stable.confirm.ch/).

> [!important] Product development
> This project will be used to develop & maintain Typr "the product".  
> It will build & push the `typr` & `typr-docs` Docker images, and deploy the official [documentation](https://typr.stable.confirm.ch/).
>
> In our infrastructure, Typr will be deployed in the dedicated [Typr infrastructure project](https://gitlab.confirm.ch/confirm/infrastructure/typr).
